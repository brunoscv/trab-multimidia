# Trabalho de Multimidia

Trabalho final da materia de multimidia V

## Getting Started

Cada imagem é um resultado dos passos requeridos

### Installing

Basta instalar as bibliotecas

```
pip install opencv-contrib-python
```

E depois

```
pip install mahotas
```
